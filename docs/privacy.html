Tether – Privacy Policy

Last updated: April 27, 2026

Overview

Tether is a Chrome extension that pops browser tabs into floating popup windows and snaps them back. It collects no personal data and makes no network requests.

Data we collect

None. Tether does not collect, transmit, or share any information about you or your browsing activity.

Permissions explained

  • tabs, windows — required to move tabs between windows.
  • storage — used locally to remember popup positions between sessions. Nothing is sent off-device.
  • scripting — used to inject a small color indicator into popup tab titles. No page content is read or stored.
  • notifications — used to show an undo prompt when a tab is auto-popped. No notification content is transmitted.
  • Host permissions (<all_urls>) — required by Chrome for the scripting permission to work on any tab, regardless of URL. Tether does not read, record, or transmit any page content.

Third parties

Tether does not use any third-party services, analytics, or advertising.
